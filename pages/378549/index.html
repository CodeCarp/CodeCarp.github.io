# 数组的概述

**1.数组的理解：**是多个相同类型的数据按一定顺序排列的集合，并使用一个名字命名，通过编号的方式对这些数据进行统一管理。

常见概念：

数组名、元素、下标、索引、数组的长度 数组 . length（存储的元素的个数）

（1）数组本身是引用数据类型，而数组的元素可以是任意类型（基本数据类型+引用数据类型）

（2）数据是在内存中开辟一段连续的空间，不连续的称为链表。

（3）数组的长度一旦确定就不在能修改。

（4）数组的分类：按维数分为一维，二维···

按数组元素的类型：基本数据类型元素的数组，引用数据类型元素的数组。

        Scanner sc = new Scanner(System.in);
        System.out.println("请输入杨辉三角的行数：");
        int hangShu =sc.nextInt();
        int[][] array = new int[hangShu][];
        for (int i = 0; i < array.length; i++) {
            array[i] = new int[i+1];// 给每一行添加一个一维数组，该数组随着 i 不断增长

                    array[i][j]=array[i-1][j-1]+array[i-1][j];

                }
            }
        }

        for (int i = 0; i < array.length; i++) {
            for (int j = 0; j < array[i].length; j++) {
                System.out.print(array[i][j]+" ");
            }
            System.out.println();
        }
    }
}

题型：定义一个int 型一维数组，包括10个元素，分别赋一些随机数，然后求出所有元素的最大值、最小值、和，并输出

package Demo02;

public class DemoTest01 {
    public static void main(String[] args) {
        int[] array = new int[10];
        for (int i = 0; i < array.length; i++) {
            array[i]=10+(int)(Math.random()*90);
            System.out.print(array[i]+" ");
        }
        int max=array[0];
        int min=array[0];
        int sum=array[0];
        for (int i = 1; i < array.length; i++) {
            max=max>array[i]?max:array[i];
            min=min<array[i]?min:array[i];
            sum+=array[i];
        }
        System.out.println("\\n"+"最大值："+max+"最小值："+min+"总和："+sum);

    }
}

4.数组元素的排序算法

衡量排序优劣的指标：1.时间复杂度 2.空间复杂度 3.稳定性

十种内部排序算法

  • 选择排序：
    • 直接选择排序、堆排序（思想）——堆排序（英语:Heapsort）：先把数组构造成一个大堆顶（父亲节点大于其子节点），然后把堆顶（数组最大值，数组第一个元素）和数组最后一个元素交换，这样就把最大值放到了数组最后边。把数组长度n-1，在进行构造堆，把剩余的第二大值放到堆顶，输出堆顶（放到剩余未排序数组后面）。以此类推，直至数组排序完成。
  • 交换排序：
    • 冒泡排序、快速排序
  • 插入排序：
    • 直接插入排序、折插入排序、Shell排序
  • 归并排序（思想）——分治思想
  • 桶式排序
  • 基数排序

# 冒泡排序（nlog2n）

基本思想：

通过对待排序序列从前往后，依次比较相邻元素的排序码，若发现逆序则交换，使排序码较大的元素逐渐从前部移向后部。

package com.sort;
/*
 * 冒泡排序
 */
import java.util.Arrays;

public class Sort {

	public static void main(String[] args) {
		int[] arr = new int[] { 43, 32, 76, -98, 0, 64, 33, -21, 32, 99 };
		int point = arr.length - 1;
		for (int j = 0; j <= point-1; j++) {
			for (int i = 0; i < point -j; i++) {// 每排序一次就确定一个最大值，排序就少一次  -j
				if (arr[i] > arr[i + 1]) {
					int temp = arr[i];
					arr[i] = arr[i + 1];
					arr[i + 1] = temp;
				}
			}
		}

		System.out.println(Arrays.toString(arr));
	}

}
