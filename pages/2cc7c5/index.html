第四章 文件目录指令

# 第四章 文件目录指令

# 4.1 文件目录类

  • pwd指令：显示当前工作目录的绝对路径

  • ls [选项] [目录或是文件]指令：

    • 常用选项：

      -a：显示当前目录所有的文件和目录，包含隐藏的；

      -l：以列表的方式显示信息。

  • cd [参数]指令： 切换到指定目录

    • cd ~：回到自己的家目录比如你是root那么当你cd ~时，你将回到/root目录
    • cd..：回到

# 4.2 创建目录

  • mkdir [选项] 要创建的目录指令：用户创建目录

    • 常用选项

      -p：创建多级目录。

注意：mkdir默认只创建一级目录，无法创建多级目录。

# 4.3 删除目录

  • rmdir指令：删除空目录
  • 如果目录下有内容时无法删除。如果要删除非空目录，需要使用 rm -rf 目录名

# 4.4 创建空文件

  • touch 文件名称指令：创建空文件

# 4.5 复制指令

cp指令：拷贝文件到指定目录

基本语法：cp [选项] source dest

  • 常用选项：

    -r：递归复制整个文件夹

案例：将/home/hello.txt拷贝到 /home/bbb目录下

案例：递归复制整个文件夹，举例。比如将 /home/bbb整个目录，拷贝到 /opt cp -r /home/bbb /opt

# 4.6 移除文件或目录

  • rm指令：移除文件或目录。

  • 基本语法：rm [选项] 文件名/目录

  • 常用选项：

    -r：递归删除整个文件夹

    -f：强制删除不提示

案例：将 /home/hello.txt删除

案例：递归删除整个文件夹 /home/bbb

# 4.7 移动文件/重命名

mv指令：mv移动文件与目录或重命名。

基本语法：

  • mv oldNameFile newNameFile（功能描述：重命名）
  • mv /temp/movefile /targetFolder （功能描述：移动文件）

# 4.8 查看文件内容

cat指令：

基本语法：

  • cat [选项] 要查看的文件

常用选项：

  • -n：显示行号

# 4.9 more指令

more指令，是一个基于VI编辑器的文本过滤器，他以全屏幕的方式按页显示文本文件的内容。more指令中内置了若干快捷键。

基本语法：

  • more 要查看的文件
操作 功能说明
空格键（space） 代表向下翻一页
Enter 代表向下翻一行
q 代表立刻离开more，不再显示该文件内容
Ctrl + F 向下滚动一屏
Ctrl + B 返回上一屏
= 输出当前行的行号
:f 输出文件名和当前行的行号

# 4.10 less指令

less指令用来分屏查看文件内容，他的功能与more相似，less指令在显示文件内容时，并不是一次将整个文件加载之后才显示，而是根据显示需要加载内容，对于显示大型文件具有较高的效率。

基本语法：less 文件名

操作 功能说明
空格键 向下翻动一页
pagedown 向下滚动一页
pageup 向上翻动一页
/字符串 向下搜寻；n向下查找，N向上查找
？字符串 向上搜寻；n向上查找，N向下查找
q 离开less状态

# 4.11 echo指令

echo指令：输出内容到控制台。

语法：echo [选项] [输出内容]

案例：使用echo指令输出环境变量，比如输出$PATH ，$HOSTNAME ，echo $HOSTNAME

# 4.12 head/tail指令

tail指令输出文件中尾部的内容，默认情况下tail指令显示文件的后10行内容。

语法：

  • tail 文件 （查看文件尾10行内容）
  • tail -n 5 文件 （查看文件尾5行内容）
  • tail -f 文件 （实时追踪该文档的所有更新）

# 4.13 输出重定向和追加

>指令（重定向）<< 追加

语法：

  • ls -l > 文件 （列表的内容写入文件a.txt中（覆盖））
  • ls -al >> 文件 （列表的内容追加到文件aa.txt的末尾）
  • cat 文件1 > 文件2 （将文件1 的内容覆盖到文件2）
  • echo “内容 ” >> 文件

# 4.14 软连接

ln指令：软连接也称为符号连接，类似于Windows里的快捷方式，主要存放了连接其他文件的路径

语法：in -s [原文件或目录] [软连接名]

ln -s /root /home/myroot在/home目录下创建一个软连接myroot，连接到/root目录下，

rm /home/myroot删除软连接
