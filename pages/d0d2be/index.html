判断是否为数字（可选小数位）

# 判断是否为数字（可选小数位）

# 解释说明

valueNew 是传入的数字或字符串。int 是指定的小数位数。

  1. 如果传入的 valueNew 为空、null 或 undefined，返回空。
  2. 将输入的 valueNew 转换为字符串形式。
  3. 使用正则表达式清除除数字和小数点外的所有字符。
  4. 清除多余的小数点，只保留第一个小数点。
  5. 将第一个小数点用特殊字符 "$#$" 替换，然后清除所有小数点。
  6. 重新将 "$#$" 替换为小数点，保留小数点后两位。
  7. 删除字符串开头的点（小数点开头的情况）。
  8. 处理整数部分和小数部分：
    • 如果没有小数点且不为空，则截取整数部分并限制长度为 int，然后转换为浮点数。
    • 如果有小数点：
      • 如果整数部分为空，则整数部分为0。
      • 如果整数部分不为空，截取整数部分并限制长度为 int。
      • 小数部分不受限制，保留原样。
  9. 返回处理后的结果。

# 代码实现

function formatDecimalNumber(valueNew, int) {
    let value = valueNew ? valueNew.toString() : '';
    if (value !== null && value !== undefined) {
        value = value.replace(/[^\d.]/g, "");
        value = value.replace(/\.{2,}/g, ".");
        value = value.replace(".", "$#$").replace(/\./g, "").replace("$#$", ".");
        value = value.replace(/^(\-)*(\d+)\.(\d{0,2}).*$/, '$1$2.$3');
        value = value.replace(/^[.]/, "");
        
        if (value.indexOf(".") < 0 && value !== "") {
            value = value.split('.')[0].substring(0, int);
            value = parseFloat(value);
        } else {
            if (value == '') value = ''
            else value = value.split('.')[0].substring(0, int) + '.' + (value.split('.')[1] ? value.split('.')[1] : '')
        }
        
        return value;
    } else {
        return null;
    }
}
