# 18. JS判断空值、0、false等

封装方法，判断空值、0、false等，适用于已定义的变量

# 18.1 代码实现

function empty(e) {
  switch (e) {
    case "":
    case 0:
    case "0":
    case null:
    case false:
    case undefined:
      return true;
    default:
      return false;
  }
}
