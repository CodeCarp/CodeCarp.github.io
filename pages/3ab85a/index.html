# 4、Spring对IOC的实现

# 4.1 IOC控制反转

  • 控制反转是一种思想
  • 控制反转是为了降低程序耦合度，提供程序扩展力，达到OCP原则，达到DIP原则
  • 控制反转，反转的是什么？
    • 将对象的创建权力交出去，给第三方容器负责
    • 将对象和对象之间关系的维护权交出去，给第三方容器负责
  • 控制反转这种思想是如何实现的？DI，依赖注入

# 4.2 依赖注入

  • 依赖注入实现了控制反转的思想
  • Spring通过依赖注入的方式来完成Bean管理的
  • Bean管理说的是：Bean对象的创建，以及Bean对象中的属性的赋值（Bean对象之间关系的维护）
  • 依赖注入：
    • 依赖指的是对象和对象之间的关联方式
    • 注入指的是一种数据传递行为，通过注入行为来让对象和对象产生关系
  • 依赖注入常见的实现方式：
    • 第一种：set注入
    • 第二种：构造器注入
    • 两者的区别：set注入是在对象已经创建出来了，再进行的注入，构造器是在对象实例化的过程中完成注入的

# 4.2.1 set注入

  • set注入，基于set方法实现的，底层会通过反射机制调用属性对应的set方法然后给属性赋值，这种方式要求属性必须对外提供set方法。

    public static boolean isSimpleValueType(Class<?> type) {
		return (Void.class != type && void.class != type &&
				(ClassUtils.isPrimitiveOrWrapper(type) || // 基本数据类型 基本数据类型对应的包装类
				Enum.class.isAssignableFrom(type) || // Enum子类
				CharSequence.class.isAssignableFrom(type) || // String或其他的CharSequence子类
				Number.class.isAssignableFrom(type) || // Number子类
				Date.class.isAssignableFrom(type) || // Date子类（实际开发中不会吧Date当做简单类型，格式不好记）
				Temporal.class.isAssignableFrom(type) || // Temporal子类（时间与时区）
				URI.class == type || // URI
				URL.class == type || // URL
				Locale.class == type || // Locale 语言类
				Class.class == type)); // Class
	}

# 4.3.4 级联属性赋值（了解）

public class Student {

    private String name;

    private Clazz clazz; // 学生属于哪个班级

    public void setClazz(Clazz clazz) {
        this.clazz = clazz;
    }

    /**
     * 使用级联属性赋值，这里需要提供get方法
     * @return
     */
    public Clazz getClazz() {
        return clazz;
    }

    public void setName(String name) {
        this.name = name;
    }

    @Override
    public String toString() {
        return "Student{" +
                "name='" + name + '\'' +
                ", clazz=" + clazz +
                '}';
    }
}

# 4.4 p命名空间注入

  • 目的：简化配置
  • xmlns 表示是该 XML 文件的默认命名空间；
  • xmlns:xsi 表示该 XML 文件遵守 xml 规范；
  • xsi:schemaLocation 表示具体用到的 schema 资源。
  • 使用p命名空间注入的前提条件包括两个：
    • 第一：在XML头部信息中添加p命名空间的配置信息：xmlnsp:p="http://www.springframeword.org/schema/p"
    • 第二：p命名空间注入是基于setter方法的，所以需要对应的属性提供setter方法
public class Dog {

    private String name;

    private int age;

        this.age = age;
    }

    public void setBirth(Date birth) {
        this.birth = birth;
    }

    @Override
    public String toString() {
        return "Dog{" +
                "name='" + name + '\'' +
                ", age=" + age +
                ", birth=" + birth +
                '}';
    }
}
