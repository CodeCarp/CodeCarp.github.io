CSS 圆角边框

# CSS 圆角边框

在CSS三中，新增了圆角边框  样式，border-radius  属性用于设置元素的外边框圆角

border-radius: 1px;
1

**radius半径（圆的半径）原理：**圆与边框的交集形成圆角的效果

  • 参数值可以为数值或百分比的形式
  • 如果是正方形，想要设置一个圆，把 数值修改为高度或宽度的一半 即可，或者直接写为 50%
  • 如果是个矩形，设置为高度的一半就可以做 
  • 该属性是一个简写属性，可以跟四个值，分别代表 左上角、右上角、右下角、左下角
  • 分开写： border-top-left-radius、border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius
