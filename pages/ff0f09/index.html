ServletConfig接口详解

# 8. ServletConfig接口详解

# 8.1 ServletConfig对象是什么？

  • Jakarta.servlet.ServletConfig
  • 显然ServletConfig是Servlet规范中的一员
  • ServletConfig是一个接口。（jakarta.servlet.Servlet是一个接口）

# 8.2 谁去实现这个接口呢？WEB服务器实现了

public class org.apache/catalina.core.StandardWrapperFacade implements ServletConfig{}
1
  • Tomcat服务器实现了ServletConfig接口
  • 思考：如果把Tomcat服务器换成jetty服务器，输出ServletConfig对象的时候，还是是这个结果吗？
    • 不一定一样、包名类名可能不一样，但是他们都实现了ServletConfig这个规范

# 8.3 一个Servlet对象中有一个ServletConfig对象。（Servlet和ServletConfig对象是一对一的）

  • 100个Servlet就有100个ServletConfig对象

# 8.4 ServletConfig对象是谁创建的？在什么时候创建的？

  • Tomcat服务器（WEB服务器创建了ServletConfig对象）
  • 在创建Servlet对象的时候，同时创建了ServletConfig对象，都在用户发送第一次请求的时候创建

# 8.5 ServletConfig接口是干啥的？有什么用？

  • ServletConfig对象：Servlet对象的配置信息对象
  • 一个Servlet对象就有一个配置信息对象
  • 两个Servlet对象就有两个配置信息对象

# 8.6 ServletConfig到底包装了什么配置信息？

  • <servlet></servlet>
  • ServletConfig对象中的包装信息是：web.xml文件中<servlet></servlet>标签的配置信息
  • Tomcat解析web.xml文件，将wen.xml文件中<servlet></servlet>标签中的配置信息自动包装到ServletConfig中

# 8.7 ServletConfig接口有哪些常用方法？

public String getInitParameter(String name); // 通过初始化参数的name获取value
public Enumeration<String> getInitParameterNames(); // 获取所有的初始化参数的name
public ServletContext getServletContext(); // 获取ServletContext对象
public String getServletName(); // 获取Servlet的name

对这些方法的使用：

public class ServletTestConfig extends GenericServlet {
    @Override
    public void service(ServletRequest request, ServletResponse response) throws ServletException, IOException {
        PrintWriter out = response.getWriter();
        response.setContentType("text/html");

        ServletConfig config = this.getServletConfig();
        // 实际上，获取一个Servlet的初始化信息可以不使用ServletConfig对象，可以通过this直接获取
        /** 在GenericServlet中
         *      public String getInitParameter(String name) {
         *         return this.getServletConfig().getInitParameter(name);
         *     }
         *
         *     public Enumeration<String> getInitParameterNames() {
         *         return this.getServletConfig().getInitParameterNames();
         *     }
         */
       //  this.getInitParameter(name)
       // this.getInitParameterNames();
        String servletName = config.getServletName();
        out.print(servletName);

        Enumeration<String> initParameterNames = config.getInitParameterNames();
        while (initParameterNames.hasMoreElements()) {
            String name = initParameterNames.nextElement();
            String initParameter = config.getInitParameter(name);// 通过name去获取value
            out.print(name + "=" + initParameter);
            out.print("<br/>");
        }

    }
}
