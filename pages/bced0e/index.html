显示具有父子关系的字符串（使用“隔开”）

# 显示具有父子关系的字符串（使用“/隔开”）

# 解释说明

  1. array 是传入的数组，其中包含要连接的元素。
  2. 函数开始时初始化一个空字符串 newStr，用于存储最终的拼接结果。
  3. 首先检查数组的长度：
    • 如果数组长度为 1，直接将数组中唯一的元素赋给 newStr。
    • 如果数组长度大于 1，则通过循环遍历数组，将每个元素用 / 分隔，添加到 newStr 中。
  4. 循环结束后，返回拼接好的字符串 newStr。

# 代码实现

function displayType(array) {
    let newStr = ""
    if(array.length === 1) newStr = array[0]
    else{
        for(let i = 0; i < array.length; i++){
            if(i !== array.length - 1) newStr += (array[i] + '/')
            else newStr += array[i]
        }
    }
    return newStr
}
