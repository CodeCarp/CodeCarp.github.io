# 9.1.3 第三种方式(附带延迟加载)

  • 多对一映射的第三种方式：两条sql语句，分步查询（这种方式常用：有点一是可复用，有点二是支持懒加载即延迟加载）

  • 什么是延迟加载：用到的时候在执行查询语句，不用的时候不查询

  • 在MyBatis中怎么开启懒加载？ association标签中添加fetchType="lazy"，这种方式是局部设置，只对当前语句起作用。fetchType="eager" ，关闭延迟加载

  • 在实际开发中是：在全局中设置延迟加载，然后在不需要使用延迟加载的地方使用fetchType="eager"

        <!--全局设置懒加载-->
    <settings>
        <setting name="lazyLoadingEnabled" value="true"/>
    </settings>

# 9.2 一对多映射

  • “一”在前，那么“一”就是主表，“多”就是副表
  • 两种实现方式：
    • 第一种方式：collection
    • 第二种方式：分步查询（支持懒加载）

# 9.2.1 使用collection方式

    /**
     * 根据班级标号查询班级信息
     * @param cid
     * @return
     */
    Clazz selectByCollection(Integer cid);

    <resultMap id="clazzResultMap" type="clazz">
        <id property="cid" column="cid"/>
        <result property="cname" column="cname"/>
        <!--一对多，使用collection-->
        <!--ofType：用于指定集合中元素的类型-->
        <collection property="stus" ofType="student">
            <id property="sid" column="sid"/>
            <result property="sname" column="sname"/>
        </collection>
    </resultMap>
    <select id="selectByCollection" resultMap="clazzResultMap">
        select c.cid,c.cname,s.sid,s.sname from t_clazz c left join t_stu s on c.cid = s.cid where c.cid = #{cid}
    </select>
