# 面向对象

# 一、类与对象Class and Object

先有类型，再有对象

对象是一个整体，拥有状态（属性值，用数据来描述，描述静态特性）

对象拥有动态特性，拥有动态行为。

对象是某一个类型的实例

面向对象的三大特性：封装，继承，多态。（抽象）

面向过程：强调的是功能行为，以函数为最小单位，考虑怎么做。

面向对象：强调具备功能的对象，以类/对象为最小单位，考虑谁来做。

# 1.1 权限修饰符

可以用来进一步实现封装特性，控制类成员的可见性。

public 共有的 可见范围最大
protected 受保护的 在无修饰符基础上（包内可以访问），添加子类可以访问
无（不写） 友好访问修饰符，包修饰符 包内可以访问，类内部可以访问
private 私有的 可见范围最小，只有类内部才能访问

# 1.2 Static与final关键字

Static

static：静态的，隶属于类

类成员：是属于类，并不是属于某一个对象

使用类成员的时候，并不需要创建对象

静态方法中，如果要访问类成员，只能访问静态成员。

普通方法中，是可以使用静态成员的

final

final关键字：用于定义常量

  1. final修饰类成员：表示成员是最终的，常量，不可改变。
  2. final可以修饰方法，此方法无法被重写
  3. final可以修饰类，此类不能被继承
