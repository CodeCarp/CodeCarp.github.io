字符串格式转换为日期格式

# 字符串格式转换为日期格式

# 解释说明

用于将格式为 "yyyy-mm-dd"的日期字符串转换为JavaScript 的 Date对象。

  1. datetimeStr 是传入的日期字符串，格式为"yyyy-mm-dd"。
  2. 使用 split("-") 方法将日期字符串拆分为年、月、日的数组。
  3. 将拆分后的年、月、日部分转换为整数，因为它们会被传递给 Date 构造函数。
  4. 由于 JavaScript 中月份是从 0 开始计数的，所以从月份部分减去 1。
  5. 使用 new Date(year, month, day) 构造函数创建一个 JavaScript 的 Date 对象，表示输入的日期。
  6. 返回这个新创建的 Date 对象。

# 代码实现

function convertStrToDate(datetimeStr) {
    let dataPart = datetimeStr.split("-");//yyyy/mm/dd格式时间
    let year = Math.floor(dataPart[0]);
    let month = Math.floor(dataPart[1]) - 1;
    let day = Math.floor(dataPart[2]);
    return new Date(year, month, day);
}
