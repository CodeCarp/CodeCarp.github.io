# 1.1 Linux概述

# 1.1.1 Linux介绍

Linux，全称GNU/Linux，是一种免费使用和自由传播的类UNIX (opens new window)操作系统，其内核由林纳斯·本纳第克特·托瓦兹 (opens new window)于1991年10月5日首次发布，它主要受到Minix (opens new window)和Unix思想的启发，是一个基于POSIX (opens new window)的多用户、多任务 (opens new window)、支持多线程 (opens new window)和多CPU (opens new window)的操作系统。它能运行主要的Unix (opens new window)工具软件、应用程序和网络协议。它支持32位 (opens new window)和64位 (opens new window)硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。Linux有上百种不同的发行版，如基于社区开发的debian (opens new window)、archlinux (opens new window)，和基于商业开发的Red Hat Enterprise Linux (opens new window)、SUSE (opens new window)、Oracle Linux (opens new window)等。

# 1.1.2 作者介绍

Linux之父——Linus Torvalds

git创作者、世界著名黑客

# 1.1.3 Linux主要的发行版

Ubuntu（乌班图）、RedHat（红帽）、CentOS、Debain[蝶变]、Fedora、SuSE、OpenSUSE

# 1.1.4 Linux的应用领域

个人桌面领域、服务器领域（最主要的领域）、嵌入式领域

# 1.1.5 Linux和Unix的关系

Unix是Linux之父，Linux衍生于Unix。

# 1.4 安装vmtools

  1. 进入centos;
  2. 点击vm菜单 ->install vmware tools
  3. centos会出现一个VM的安装包，xx.tar.gz
  4. 拷贝到 /opt
  5. 使用解压命令tar，得到一个安装文件 cd /opt
  6. 进入该vm解压的目录， /opt目录下；tar -zxvf xxx.tar.gz
  7. 安装 ./vmware-install.pl
  8. 全部使用默认设置即可，就可以安装成功
  9. 注意：安装vmtools需要有gcc

# 1.5 Linux目录结构

/bin【常用】（/user/bin,/user/local/bin）：是Binary的缩写，这个目录存放着最经常使用的命令；

/sbin（/user/sbin,/usr/local/sbin）：S就是Super User的意思，这里存放着的是系统管理员使用的系统管理程序；

/home【常用】：存放普通用户的主目录，在linux中每个用户都有一个自己的目录，一般该目录是以用户的账号命名的；

/root【常用】：该目录为系统管理员，也称为超级权限者的用户主目录；

/lib：系统开机所需最基本的动态连接共享库，其作用类似Windows里的DLL文件，几乎所有的应用程序都需要用到这些共享库；

/lost+found：这个目录一般情况下是空的，当系统非法关机后，这里就存放了一些文件。

/etc【常用】：所有的系统管理所需要的配置文件和子目录，比如安装mysql数据库my.conf

/usr【常用】：这是一个非常重要的目录，用户的很多应用程序和文件都放在这份目录下，类似于window下的program files目录；

/boot【常用】：存放的是启动Linux时使用的一些核心文件，包括一些连接文件以及镜像文件

/proc：这个目录是一个虚拟的目录，他是系统内存的映射，访问这个目录来获取系统的信息

/srv：service缩写，该目录存放一些服务启动之后需要提取的数据

/sys：这是Linux2.6内核的一个很大变化，该目录下安装了2.6内核中新出现的文件系统sysfs

/tmp：这个目录是用来存放一些临时文件的；

# 1.7 vi和vim的基本介绍

Linux系统内置vi文本编辑器

Vim具有程序编辑的能力，可以看做的vi的增强版。

# 1.7.1 vi和vim常用的三种模式

  • 正常模式：

    以vim打开骗一个档案就直接进入一般模式了（默认的模式），在这个模式中，你可以使用【上下左右】按钮来移动光标，可以使用【删除字符】或【删除整行】来处理档案内容，也可以使用【复制、粘贴】来处理文件数据。

  • 插入模式：

    按下i、I、o、O、a、A、r、R等任何一个字母之后才会进入编辑模式，一般来说按下i即可。

  • 命令行模式：

    这个模式中，可以提供你相关指令，完成读取、存盘、替换、离开vim、显示行号等的动作则是在此模式中达成的。

# 1.7.3 vi和vim快捷键

  1. 拷贝当前行——yy、拷贝当前行向下的5行——5yy、并粘贴（p）；
  2. 删除当前行——dd、删除当前行向下的5行——5dd；
  3. 在文件中查找某个单词【命令行下 /关键字、回车 查找，输入 n 就是查找下一个；
  4. 设置文件的行号，取消文件的行号：【命令行下 :set nu 和 :set nonu】
  5. 快速定位到文件的最末行【G】和最首行【gg】
  6. 在一个文件中输入“hello”在一般模式下，然后有撤销这个动作【u】；
  7. 在光标移动到指定行：在一般模式下，输入行号，在输入shift+g，【5 shift+g】；
  8. 更多的看文档。

# 1.8 关机&重启指令

基本介绍：

  • shutdown -h now立刻关机

  • shutdown -h 1一分钟后关机

  • shutdown -r now立刻重启

  • halt关机

  • reboot重启

  • sync把内存的数据同步到磁盘

    注意：不管是重启还是关机，首先要运行sync指令，把内存中的数据写到磁盘中

    目前的shutdown/reboot/halt等指令均已经在关机前进行了sync，但最好还是自行运行一下sync命令。
